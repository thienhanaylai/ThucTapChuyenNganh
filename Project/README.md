# ThucTapChuyenNganh ExpressJS

Đồ án thực tập chuyên ngành xây dựng ứng dụng web sử dụng NodeJS (Express) và MongoDB.

**Thông tin sinh viên:**

- **Họ tên:** Ngô Gia Bảo
- **MSSV:** DH52200363

---

## Yêu cầu

Trước khi cài đặt, đảm bảo máy tính của bạn đã cài đặt các công cụ sau:

1.  **Node.js**: v22.19.0
2.  **MongoDB**: Phải cài đặt MongoDB Community Server và đảm bảo service MongoDB đang chạy ở background.

## Hướng dẫn cài đặt

**Bước 1: Clone dự án về máy**

```bash
git clone (https://github.com/thienhanaylai/ThucTapChuyenNganh)
cd ThucTapChuyenNganh
```

**Bước 2: Clone dự án về máy**

```bash
npm install
```

**Bước 3: Clone dự án về máy**

```bash
npm run dev
```
